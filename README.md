# vue internationalization demo

#### A vue application to demonstrate implementation of localization in Vue using I18n package.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files
